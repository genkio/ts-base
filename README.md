# ts-base

> Just a TypeScript project starter

## Project setup

    npm install

### Compiles

    npm run build

### Run your tests

    npm t
    npm run test-watch

### Lints and fixes files

> Set `tslint.autoFixOnSave` to true for auto fix

    npm run tslint
    npm run tslint | grep ERROR

### Format / prettify

    npm run format
